# 키보드 & 마우스 단축키

ReciPro는 버튼이나 메뉴에는 보이지 않는 많은 기능을 **키 조합**과 **수정 키와 결합한 마우스 버튼**에 연결합니다. 이 페이지는 그것들을 모두 한곳에 모았습니다. 각 창의 페이지에서도 해당 단축키를 맨 위 부근에 다시 정리해 두었습니다.

<kbd>F1</kbd>은 **모든** 창에서 동작하며, 해당 창에 대한 이 온라인 매뉴얼 페이지를 엽니다.

---

## 응용 프로그램 전역 단축키

이 단축키들은 [메인 창](0-main-window.md)이 등록하지만, 구조 뷰어, 스테레오넷, 회절 시뮬레이터, Spot ID, 계산기 창에 포커스가 있는 동안에도 활성 상태를 유지합니다.

| 단축키 | 동작 |
|----------|--------|
| <kbd>CTRL</kbd>+<kbd>SHIFT</kbd>+<kbd>D</kbd> | 회절 시뮬레이터 켜기/끄기 |
| <kbd>CTRL</kbd>+<kbd>SHIFT</kbd>+<kbd>V</kbd> | 구조 뷰어 켜기/끄기 |
| <kbd>CTRL</kbd>+<kbd>SHIFT</kbd>+<kbd>S</kbd> | 스테레오넷 켜기/끄기 |
| <kbd>CTRL</kbd>+<kbd>SHIFT</kbd>+<kbd>T</kbd> | Spot ID 켜기/끄기 |
| <kbd>CTRL</kbd>+<kbd>SHIFT</kbd> + 화살표 키 | 결정을 그 방향으로 한 단계 회전 (두 화살표를 동시에 누르면 대각선) |
| <kbd>CTRL</kbd> 빠르게 두 번 누르기 | 계산기 켜기/끄기 |
| <kbd>CTRL</kbd>+<kbd>SHIFT</kbd>+<kbd>R</kbd> | 선택한 결정의 *Reserved* 플래그 전환 |
| <kbd>CTRL</kbd>+<kbd>ALT</kbd>+<kbd>SHIFT</kbd>+<kbd>C</kbd> | GUI 스크린샷 캡처 (개발자 도구; 먼저 **GUI 구성 요소 캡처**를 활성화) |
| ReciPro 시작 중 <kbd>CTRL</kbd> 누른 채로 유지 | OpenGL을 비활성화하고 시작 (그래픽 문제 복구용) |

## 공통 상호작용 모델

ReciPro의 거의 모든 상호작용 뷰는 세 가지 계열 중 하나에 속합니다. 어느 계열인지 알면 각 창을 일일이 외우지 않고도 끌기/확대 동작을 알 수 있습니다.

### 3-D OpenGL 뷰 { #3d }

[구조 뷰어](5-structure-viewer.md), [회전 기하학](4-rotation-geometry.md), [스테레오넷](6-stereonet.md)의 3-D 구, [전자 궤적](8-electron-trajectory.md), [EBSD](12-ebsd-simulation.md)의 기하/master pattern 뷰에서 사용됩니다.

| 동작 | 결과 |
|--------|--------|
| 왼쪽 끌기 | 회전 — 중심 부근은 트랙볼, 가장자리 부근은 면내 굴림 |
| 오른쪽 위/아래로 끌기, 또는 마우스 휠 | 확대/축소 |
| 가운데 끌기 | 이동 (활성화된 곳에서만) |
| <kbd>CTRL</kbd> + 오른쪽 위/아래로 끌기 | 카메라 거리 변경 (원근 모드에서만) |
| <kbd>CTRL</kbd> + 오른쪽 더블 클릭 | 정사영 / 원근 투영 전환 |

개별 창에서는 이동 또는 확대/축소를 끌 수 있습니다 (예를 들어 전자 궤적과 EBSD 3-D 뷰에서는 이동이 비활성화되어 있습니다).

### 회절 패턴 뷰 { #pattern }

[회절 시뮬레이터](7-diffraction-simulator/index.md)의 패턴, [EBSD](12-ebsd-simulation.md)의 키쿠치 패턴, 2-D [스테레오넷](6-stereonet.md)에서 사용됩니다. 3-D 뷰와의 결정적 차이: **끌기가 카메라만이 아니라 결정 자체를 회전시킵니다**. 따라서 연동된 모든 창이 함께 업데이트됩니다.

| 동작 | 결과 |
|--------|--------|
| 중심 부근에서 왼쪽 끌기 | 결정을 기울이기 |
| 바깥 영역에서 왼쪽 끌기 | 시선축/빔축을 중심으로 결정 회전 |
| 마우스 오른쪽 클릭 | 축소 |
| 오른쪽으로 사각형 끌기 | 선택한 영역으로 확대 |
| 가운데 끌기 | 이동 |

이 뷰에는 마우스 휠 확대/축소가 **없습니다**.

### 이미지 뷰 { #image }

[HRTEM/STEM](9-hrtem-stem-simulator/index.md) 결과 창, [Spot ID v2](11-spot-id-v2.md)의 이미지, [EBSD](12-ebsd-simulation.md)의 2-D master pattern에서 사용됩니다.

| 동작 | 결과 |
|--------|--------|
| 왼쪽 끌기 / 가운데 끌기 | 이동 |
| 마우스 휠 위 / 아래 | 커서 위치에서 확대 (×2) / 축소 (×0.5) |
| 오른쪽으로 사각형 끌기 | 선택한 영역으로 확대 |
| 마우스 오른쪽 클릭 / 오른쪽 더블 클릭 | 축소 (×0.5) |

## 창별 참조

### 0. 메인 창
[페이지 열기 →](0-main-window.md) · 위의 응용 프로그램 전역 단축키도 함께 참조.

| 단축키 | 동작 |
|----------|--------|
| 방위 위젯(왼쪽 아래) 왼쪽 끌기 | 결정 회전 |
| 방위 위젯 오른쪽 더블 클릭 | 위젯 이미지를 클립보드에 복사 |
| 기능 버튼 한 번 클릭 / 더블 클릭 | 해당 창 켜기/끄기 / 앞으로 가져오기 |
| 목록에서 결정 마우스 오른쪽 클릭 | 컨텍스트 메뉴 (이름 바꾸기 / 복제 / 삭제 / CIF 내보내기…) |
| **Current Index** 레이블 더블 클릭 | max-UVW 상자 표시/숨기기 |
| 파일 끌어다 놓기 | 결정 목록(`.xml`, `.cdb2`) 또는 결정(`.cif`, `.amc`) 불러오기 |

### 1. 결정 데이터베이스
[페이지 열기 →](1-crystal-database.md)

| 단축키 | 동작 |
|----------|--------|
| 검색 필드에서 <kbd>ENTER</kbd> | 검색 실행 |
| 결과 행 클릭 | 해당 결정 불러오기 |
| 주기율표 팝업에서 원소 클릭 | 필터 순환: 무시 → 반드시 포함 → 반드시 제외 |

### 2. 대칭 정보 · 3. 빔 상호작용
대칭 정보에는 특별한 키/마우스 조합이 없습니다. 빔 상호작용에서는 <kbd>F1</kbd>과 **복사** 버튼 외에, **산란 인자** 그래프의 세로 커서를 끌어서 각 원소의 값을 읽을 수 있습니다.
[대칭 정보 →](2-symmetry-information.md) · [빔 상호작용 →](3-beam-interaction.md)

### 4. 회전 기하학
[페이지 열기 →](4-rotation-geometry.md) — 여섯 개의 **연동된** [3-D 뷰](#3d); 어느 하나를 회전하면 여섯 개 모두가 함께 회전합니다. 작은 *Axes* / *Objects* 뷰는 확대/축소와 이동이 비활성화되어 있습니다.

### 5. 구조 뷰어
[페이지 열기 →](5-structure-viewer.md) — 메인 뷰는 [3-D 뷰](#3d)입니다.

| 단축키 | 동작 |
|----------|--------|
| <kbd>CTRL</kbd>+<kbd>SHIFT</kbd>+<kbd>C</kbd> | 렌더링된 이미지를 클립보드에 복사 |
| 원자 왼쪽 더블 클릭 | 좌표, 최근접 이웃 거리, 결합각 표시 |
| 결정축 기즈모 왼쪽 끌기 | 모델 회전 (면내 굴림 없음) |
| 광원 기즈모 왼쪽 끌기 | 조명 방향 변경 |

### 6. 스테레오넷
[페이지 열기 →](6-stereonet.md) — 2-D 네트는 [회절 패턴 뷰](#pattern)이고, 선택적 3-D 구는 [3-D 뷰](#3d)입니다.

| 단축키 | 동작 |
|----------|--------|
| 네트 왼쪽 더블 클릭 | **Plane**과 **Axis** 투영 전환 |
| 네트 위로 마우스 이동 | 커서 아래의 (hkl)/[uvw] 읽기 |

### 7. 회절 시뮬레이터
[페이지 열기 →](7-diffraction-simulator/index.md) — 패턴은 [회절 패턴 뷰](#pattern)입니다 (휠 확대/축소 없음).

| 단축키 | 동작 |
|----------|--------|
| 스폿 왼쪽 더블 클릭 | 반사 상세 표시 (지수, *d*, 구조 인자, 여기 오차) |
| <kbd>CTRL</kbd> + 가운데 끌기 | 검출기 중심 이동 (검출기 영역이 표시될 때) |
| 상태 표시줄 오른쪽 더블 클릭 | 현재 설정의 텍스트 요약 복사 |
| 켜진 레이어 버튼(Spots / Kikuchi / Debye / Scale) 오른쪽 더블 클릭 | 해당 레이어를 깜박이기 |
| 스테레오넷 왼쪽 더블 클릭 — **TEM 홀더** 창 | 홀더 기울기를 그 지점으로 설정 |
| 화살표 키 — **TEM 홀더** 창 | 홀더 기울기를 단계적으로 변경 (먼저 **Arrow keys** 체크) |
| `.prm` / 이미지 끌어다 놓기 — **검출기 기하학**, 또는 `.txt` — **동적 압축** | 해당 데이터 불러오기 |

### 11. Spot ID v2
[페이지 열기 →](11-spot-id-v2.md) — 이미지는 스폿 편집이 더해진 [이미지 뷰](#image)입니다.

| 단축키 | 동작 |
|----------|--------|
| 이미지 왼쪽 더블 클릭 | 스폿 추가 (피크 피팅) |
| <kbd>CTRL</kbd> + 왼쪽 더블 클릭 | 스폿을 추가하고 직접빔(000)으로 표시 |
| 스폿 왼쪽 클릭 | 가장 가까운 스폿 선택 |
| <kbd>CTRL</kbd> + 스폿 마우스 오른쪽 클릭 | 가장 가까운 스폿 삭제 |
| <kbd>CTRL</kbd> + 화살표 키 | 선택한 스폿을 1픽셀 이동 |
| 스폿 행 머리글 더블 클릭 | 해당 스폿으로 확대 (×2) |
