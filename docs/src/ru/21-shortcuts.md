# Сочетания клавиш и мыши

ReciPro связывает многие функции с **комбинациями клавиш** и с **кнопками мыши в сочетании с клавишами-модификаторами** — то, что не отображается на кнопке или в меню. На этой странице они собраны все в одном месте. Страница каждого окна также повторяет его сочетания клавиш ближе к началу.

<kbd>F1</kbd> работает в **каждом** окне и открывает соответствующую страницу этого онлайн-руководства.

---

## Сочетания клавиш для всего приложения

Они устанавливаются [главным окном](0-main-window.md), но остаются активными, пока в фокусе находятся окна Просмотр структуры, Стереосеть, Симулятор дифракции, Spot ID и Калькулятор.

| Сочетание | Действие |
|----------|--------|
| <kbd>CTRL</kbd>+<kbd>SHIFT</kbd>+<kbd>D</kbd> | Включить/выключить Симулятор дифракции |
| <kbd>CTRL</kbd>+<kbd>SHIFT</kbd>+<kbd>V</kbd> | Включить/выключить Просмотр структуры |
| <kbd>CTRL</kbd>+<kbd>SHIFT</kbd>+<kbd>S</kbd> | Включить/выключить Стереосеть |
| <kbd>CTRL</kbd>+<kbd>SHIFT</kbd>+<kbd>T</kbd> | Включить/выключить Spot ID |
| <kbd>CTRL</kbd>+<kbd>SHIFT</kbd> + клавиши со стрелками | Повернуть кристалл на один шаг в этом направлении (удерживайте две стрелки для диагонали) |
| Двойное нажатие <kbd>CTRL</kbd> | Включить/выключить Калькулятор |
| <kbd>CTRL</kbd>+<kbd>SHIFT</kbd>+<kbd>R</kbd> | Переключить признак *Reserved* выбранного кристалла |
| <kbd>CTRL</kbd>+<kbd>ALT</kbd>+<kbd>SHIFT</kbd>+<kbd>C</kbd> | Сделать снимок экрана GUI (инструмент разработчика; сначала включите **Capture GUI Components**) |
| Удерживать <kbd>CTRL</kbd> при запуске ReciPro | Запуск с отключённым OpenGL (восстановление при проблемах с графикой) |

## Общие модели взаимодействия

Почти каждое интерактивное представление в ReciPro относится к одному из трёх семейств. Зная семейство, вы знаете поведение при перетаскивании/масштабировании, не запоминая каждое окно.

### Трёхмерные представления OpenGL { #3d }

Используются в [Просмотре структуры](5-structure-viewer.md), [Геометрии вращения](4-rotation-geometry.md), 3-D-сфере [Стереосети](6-stereonet.md), [Траекториях электронов](8-electron-trajectory.md) и в представлениях геометрии / master pattern [EBSD](12-ebsd-simulation.md).

| Действие | Результат |
|--------|--------|
| Перетаскивание левой кнопкой | Поворот — трекбол около центра, вращение в плоскости около края |
| Перетаскивание правой кнопкой вверх/вниз или колесо мыши | Масштабирование |
| Перетаскивание средней кнопкой | Панорамирование (только там, где включено) |
| <kbd>CTRL</kbd> + перетаскивание правой кнопкой вверх/вниз | Изменить расстояние до камеры (только в режиме перспективы) |
| <kbd>CTRL</kbd> + двойной щелчок правой кнопкой | Переключить ортографическую / перспективную проекцию |

Отдельные окна могут отключать панорамирование или масштабирование (например, в Траекториях электронов и 3-D-представлениях EBSD панорамирование отключено).

### Представления дифракционных картин { #pattern }

Используются для картины [Симулятора дифракции](7-diffraction-simulator/index.md), картины Кикучи [EBSD](12-ebsd-simulation.md) и 2-D-[Стереосети](6-stereonet.md). Ключевое отличие от 3-D-представлений: **перетаскивание поворачивает сам кристалл**, а не только камеру, поэтому каждое связанное окно обновляется одновременно.

| Действие | Результат |
|--------|--------|
| Перетаскивание левой кнопкой около центра | Наклонить кристалл |
| Перетаскивание левой кнопкой во внешней области | Повернуть кристалл вокруг оси обзора/пучка |
| Щелчок правой кнопкой | Уменьшить масштаб |
| Перетаскивание правой кнопкой рамки | Увеличить масштаб до выбранной области |
| Перетаскивание средней кнопкой | Панорамирование |

В этих представлениях **нет** масштабирования колесом мыши.

### Представления изображений { #image }

Используются в областях результатов [HRTEM/STEM](9-hrtem-stem-simulator/index.md), для изображения [Spot ID v2](11-spot-id-v2.md) и для 2-D-master pattern [EBSD](12-ebsd-simulation.md).

| Действие | Результат |
|--------|--------|
| Перетаскивание левой / средней кнопкой | Панорамирование |
| Колесо мыши вверх / вниз | Увеличить (×2) / уменьшить (×0.5) масштаб у курсора |
| Перетаскивание правой кнопкой рамки | Увеличить масштаб до выбранной области |
| Щелчок правой кнопкой / двойной щелчок правой кнопкой | Уменьшить масштаб (×0.5) |

## Справочник по окнам

### 0. Главное окно
[Открыть страницу →](0-main-window.md) · плюс сочетания клавиш для всего приложения выше.

| Сочетание | Действие |
|----------|--------|
| Перетаскивание левой кнопкой виджета ориентации (внизу слева) | Повернуть кристалл |
| Двойной щелчок правой кнопкой по виджету ориентации | Скопировать изображение виджета в буфер обмена |
| Одиночный щелчок / двойной щелчок по функциональной кнопке | Включить/выключить это окно / вывести его на передний план |
| Щелчок правой кнопкой по кристаллу в списке | Контекстное меню (Переименовать / Дублировать / Удалить / Экспорт CIF…) |
| Двойной щелчок по подписи **Current Index** | Показать / скрыть поле max-UVW |
| Перетащить файл | Загрузить список кристаллов (`.xml`, `.cdb2`) или кристалл (`.cif`, `.amc`) |

### 1. База данных кристаллов
[Открыть страницу →](1-crystal-database.md)

| Сочетание | Действие |
|----------|--------|
| <kbd>ENTER</kbd> в поле поиска | Выполнить поиск |
| Щелчок по строке результата | Загрузить этот кристалл |
| Щелчок по элементу во всплывающей периодической таблице | Циклически менять его фильтр: игнорировать → должен содержать → должен исключать |

### 2. Сведения о симметрии · 3. Взаимодействие пучка
Сведения о симметрии не имеют особых комбинаций клавиш/мыши. Во Взаимодействии пучка, помимо <kbd>F1</kbd> и кнопок **Copy**, вертикальный курсор на графике **Scattering factors** можно перетаскивать, чтобы считывать значение каждого элемента.
[Симметрия →](2-symmetry-information.md) · [Взаимодействие пучка →](3-beam-interaction.md)

### 4. Геометрия вращения
[Открыть страницу →](4-rotation-geometry.md) — шесть **связанных** [3-D-представлений](#3d); поворот любого из них поворачивает все шесть одновременно. В небольших представлениях *Axes* / *Objects* масштабирование и панорамирование отключены.

### 5. Просмотр структуры
[Открыть страницу →](5-structure-viewer.md) — главное представление является [3-D-представлением](#3d).

| Сочетание | Действие |
|----------|--------|
| <kbd>CTRL</kbd>+<kbd>SHIFT</kbd>+<kbd>C</kbd> | Скопировать отрисованное изображение в буфер обмена |
| Двойной щелчок левой кнопкой по атому | Показать координаты, расстояния до ближайших соседей и валентные углы |
| Перетаскивание левой кнопкой гизмо кристаллических осей | Повернуть модель (без вращения в плоскости) |
| Перетаскивание левой кнопкой гизмо освещения | Изменить направление освещения |

### 6. Стереосеть
[Открыть страницу →](6-stereonet.md) — 2-D-сеть является [представлением дифракционной картины](#pattern); опциональная 3-D-сфера является [3-D-представлением](#3d).

| Сочетание | Действие |
|----------|--------|
| Двойной щелчок левой кнопкой по сети | Переключить между проекцией **Plane** и **Axis** |
| Перемещение мыши над сетью | Считать (hkl)/[uvw] под курсором |

### 7. Симулятор дифракции
[Открыть страницу →](7-diffraction-simulator/index.md) — картина является [представлением дифракционной картины](#pattern) (без масштабирования колесом).

| Сочетание | Действие |
|----------|--------|
| Двойной щелчок левой кнопкой по рефлексу | Показать сведения о рефлексе (индекс, *d*, структурный фактор, ошибка возбуждения) |
| <kbd>CTRL</kbd> + перетаскивание средней кнопкой | Переместить центр детектора (когда отображается область детектора) |
| Двойной щелчок правой кнопкой по строке состояния | Скопировать текстовую сводку текущих настроек |
| Двойной щелчок правой кнопкой по активной кнопке слоя (Spots / Kikuchi / Debye / Scale) | Заставить этот слой мигать |
| Двойной щелчок левой кнопкой по стереосети — окно **TEM holder** | Установить наклон держателя в эту точку |
| Клавиши со стрелками — окно **TEM holder** | Пошагово изменять наклон держателя (сначала отметьте **Arrow keys**) |
| Перетащить `.prm` / изображение — **Detector geometry**, или `.txt` — **Dynamic compression** | Загрузить эти данные |

[Открыть страницу →](11-spot-id-v2.md) — изображение является [представлением изображения](#image) с наложенным редактированием рефлексов.
