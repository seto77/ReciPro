## Краткое введение
* ReciPro — это бесплатное программное обеспечение под лицензией MIT, предоставляющее разнообразные кристаллографические расчёты и моделирование в электронной микроскопии.
* С момента публикации на GitHub (март 2020 г.) ReciPro был загружен в общей сложности более 27 000 раз и используется многими кристаллографами и специалистами по электронной микроскопии.

## Возможности
* **Full GUI** : Все операции выполняются через графический интерфейс. Большинство операций ввода-вывода файлов поддерживает перетаскивание (drag & drop).
* **Список кристаллов** : Управление несколькими кристаллами одновременно; нет необходимости открывать отдельное окно для каждого кристалла.
* **База данных пространственных групп** : Встроенная база данных, охватывающая 230 пространственных групп из International Tables Volume A, а также 530 символов Холла, с элементами симметрии, позициями Уайкоффа и правилами погасания. Элементы симметрии и общие положения можно изобразить в виде схематических диаграмм в стиле *International Tables* Vol. A (см. [2. Сведения о симметрии](2-symmetry-information.md)).
* **Атомная информация** : Факторы рассеяния (рентгеновские, электронные, нейтронные), характеристические энергии рентгеновского излучения, изотопные соотношения и т. д. для элементов от H (1) до Cf (98).
* **Гибкое вращение кристалла** : Задание ориентации по индексам оси зоны / плоскости кристалла или перетаскиванием мышью. Для тригональной/гексагональной систем поддерживается нотация Миллера-Браве (4-индексная *hkil*). Состояние поворота синхронизируется между всеми окнами моделирования.
* **Моделирование дифракции** : Кинематическая и динамическая (метод блоховских волн) дифракция электронов, рентгеновская дифракция (включая прецессионную и обратную камеры Лауэ), прецессионная дифракция электронов (PED) и дифракция сходящегося пучка электронов (CBED). Моделирование держателя ПЭМ связывает дифракционную картину с углами наклона держателя.
* **Моделирование HRTEM / STEM** : Моделирование изображений ПЭМ высокого разрешения с моделями частичной когерентности; STEM с тепловым диффузным рассеянием.
* **EBSD и траектории электронов** : Моделирование картин EBSD и моделирование траекторий электронов методом Монте-Карло (см. [8. Траектории электронов](8-electron-trajectory.md)).
* **Индексирование рефлексов** : Автоматическое обнаружение, аппроксимация и индексирование дифракционных рефлексов с экспериментальных изображений (Spot ID v1/v2).
* **Макрос** : Макрос с синтаксисом Python для автоматизации операций (см. [20. Макрос](20-macro/index.md)).
* **Светлая / тёмная тема** : Интерфейс следует выбираемому светлому или тёмному цветовому режиму.

## Системные требования
| Элемент | Минимум | Рекомендуется |
|------|---------|-------------|
| ОС | Windows с [.NET Desktop Runtime 10.0](https://dotnet.microsoft.com/download/dotnet/10.0) (Windows on ARM64 поддерживается) | Windows 11 |
| GPU | OpenGL 1.3 | Внешний GPU с OpenGL 4.3 |
| Память | – | 16 ГБ или более |
| CPU | – | 8+ ядер (для динамических расчётов) |

**Windows on ARM (нативная, экспериментальная)** : Экспериментальный нативный переносной пакет ARM64 (`ReciPro-v.X_arm64.zip`, self-contained — установка .NET Runtime не требуется) доступен на [странице релизов](https://github.com/seto77/ReciPro/releases/latest). Обычные пакеты x64 также работают на ARM64-Windows под встроенной эмуляцией. Замечания по настройке и ограничениям см. в разделе [Устранение неполадок](troubleshooting.md#windows-on-arm).

**macOS (неофициально)** : ReciPro официально поддерживает только Windows, однако сообщалось, что переносной ZIP-пакет **win-x64** работает на macOS (Apple Silicon) с использованием оболочки Wine Sikarugir в сочетании с драйвером OpenGL Mesa3D. Опубликованное пользователем руководство по настройке доступно по адресу <https://github.com/Ryo-fkushima/ReciPro_macOS_memo>. Обратите внимание, что этот путь официально не поддерживается и некоторые символы (Å, верхние индексы, стрелки) могут отображаться некорректно. ZIP для ARM64 **не** работает на macOS + Wine, а нынешний путь через x64 + Rosetta 2, как ожидается, перестанет работать начиная с macOS 28 (осень 2027 г.) — подробности см. в разделе [Устранение неполадок](troubleshooting.md#mac-linux).

## Как пользоваться этим руководством

Это руководство на GitHub Pages в настоящее время является основным авторитетным источником. Используйте навигацию слева для просмотра по главам или поиск в заголовке, чтобы найти имя функции или метку интерфейса. Старые PDF-руководства сохранены для архивной справки.

* **Архивный PDF (английский):** <https://raw.githubusercontent.com/seto77/ReciPro/master/ReciPro/doc/ReciProManual(en).pdf>
* **Архивный PDF (японский):** <https://raw.githubusercontent.com/seto77/ReciPro/master/ReciPro/doc/ReciProManual(ja).pdf>

## Быстрый старт
1. Загрузите и установите со страницы [Releases](https://github.com/seto77/ReciPro/releases/latest).
2. Выберите кристалл из встроенного списка (~80 кристаллов). Вы также можете импортировать файлы CIF или использовать [CSManager](https://github.com/seto77/CSManager).
3. Вызывайте функции с правой панели: Просмотр структуры, Стереосеть, Симулятор дифракции, Моделирование HRTEM и т. д.
4. Поворачивайте кристалл перетаскиванием мышью или вводом индексов оси зоны / плоскости.
