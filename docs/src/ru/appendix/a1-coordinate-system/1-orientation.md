# Приложение A1.1. Базовая система координат и ориентация кристалла

<!-- 260526Cl: 図(Coordinates1-3)に合わせ座標記号を数式(MathJax)化・用語色を図に一致 (.rp-*: docs/src/assets/stylesheets/extra.css)。 -->

На этой странице определяется **базовая (ориентационная) система координат** ReciPro, используемая везде, где задействован поворот кристалла (Главное окно, Просмотр структуры, Стереосеть, Геометрия вращения и моделирование дифракции), а также то, как выражаются начальная ориентация кристалла и поворот через углы Эйлера. Отдельная система, используемая для размещения детектора в **Симуляторе дифракции**, описана в [A1.2. Система координат для моделирования дифракции](2-diffraction.md).

---

## Определение ориентации

ReciPro использует **правую систему координат**, закреплённую относительно монитора:

| Ось | Направление |
|------|-----------|
| <span class="rp-red">$X$</span> | Вправо по монитору |
| <span class="rp-green">$Y$</span> | Вверх по монитору |
| <span class="rp-blue">$Z$</span> | Перпендикулярно из монитора, к наблюдателю |

![Координатные оси ReciPro, показанные на мониторе](../../../assets/references/Coordinates1.png){width=400px}

**Направление пучка** соответствует направлению взгляда (в глубь монитора), то есть оси <span class="rp-blue">$-Z$</span>.

Большинство операций в ReciPro затрагивают только *направления* (выражаемые как матрицы поворота 3×3) и не требуют явного начала координат. Единственное исключение — функция **Симулятор дифракции**, которой нужно явное начало координат — см. [A1.2. Система координат для моделирования дифракции](2-diffraction.md).

## Начальная ориентация кристалла

Начальная ориентация (при первом запуске или после **Сбросить поворот**) определяется так:

1. Ось <span class="rp-blue">$c$</span> совмещена с осью <span class="rp-blue">$Z$</span>.
2. Ось <span class="rp-green">$b$</span> лежит в плоскости <span class="rp-green">$Y$</span><span class="rp-blue">$Z$</span>, вблизи оси <span class="rp-green">$Y$</span>.
3. Ось <span class="rp-red">$a$</span> затем задаётся осями <span class="rp-green">$b$</span> и <span class="rp-blue">$c$</span> (правило правой руки).

![Начальная ориентация: оси кристалла a / b / c относительно X / Y / Z, с падающим пучком вдоль −Z](../../../assets/references/Coordinates2.png){width=300px}

Эквивалентно:

- Направление из монитора (к наблюдателю) — это ось зоны **[001]**.
- Направление вправо по монитору — это нормаль к плоскости **(100)**.

> **Примечание:** Ось <span class="rp-blue">$c$</span> (= [001]) всегда совпадает с <span class="rp-blue">$Z$</span>, однако в некоторых кристаллических системах оси <span class="rp-red">$a$</span> и <span class="rp-green">$b$</span> **не** обязательно совпадают с <span class="rp-red">$X$</span> и <span class="rp-green">$Y$</span>.

## Углы Эйлера

Ориентация кристалла выражается тремя углами Эйлера <span class="rp-olive">$\Phi$</span>, <span class="rp-cyan">$\theta$</span>, <span class="rp-magenta">$\Psi$</span>, применяемыми в порядке <span class="rp-blue">$Z$</span>–<span class="rp-red">$X$</span>–<span class="rp-blue">$Z$</span> (<span class="rp-magenta">$\Psi$</span>, затем <span class="rp-cyan">$\theta$</span>, затем <span class="rp-olive">$\Phi$</span>). Когда все три угла равны нулю, соответствующие оси поворота таковы:

| Угол | Ось (когда все углы = 0) | Ранг |
|-------|----------------------------|------|
| <span class="rp-olive">$\Phi$</span> | <span class="rp-blue">$Z$</span> | 1-й (высший) |
| <span class="rp-cyan">$\theta$</span> | <span class="rp-red">$X$</span> | 2-й (средний) |
| <span class="rp-magenta">$\Psi$</span> | <span class="rp-blue">$Z$</span> | 3-й (низший) |

![Оси поворота углов Эйлера — Φ (жёлтая), θ (голубая), Ψ (пурпурная) — показаны при 0° (вверху) и при 15° (внизу)](../../../assets/references/Coordinates3.png){width=400px}

Три угла образуют **иерархию**: <span class="rp-olive">$\Phi$</span> — поворот наивысшего ранга, за ним следует <span class="rp-cyan">$\theta$</span>, затем <span class="rp-magenta">$\Psi$</span>. Направление оси более низкого ранга зависит от состояния поворотов более высокого ранга. Например, при <span class="rp-olive">$\Phi$</span> = <span class="rp-cyan">$\theta$</span> = <span class="rp-magenta">$\Psi$</span> = 15° ось <span class="rp-olive">$\Phi$</span> по-прежнему совпадает с <span class="rp-blue">$Z$</span>, но оси <span class="rp-cyan">$\theta$</span> и <span class="rp-magenta">$\Psi$</span> в общем случае не совпадают ни с одной из осей <span class="rp-red">$X$</span>, <span class="rp-green">$Y$</span> или <span class="rp-blue">$Z$</span>.

> Окно **Геометрия вращения** может заново выразить эту ориентацию в произвольном, специфичном для эксперимента соглашении об углах Эйлера (например, чтобы согласовать его с лабораторным гониометром). См. [4. Геометрия вращения](../../4-rotation-geometry.md).
