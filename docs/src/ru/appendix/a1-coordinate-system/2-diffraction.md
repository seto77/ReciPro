# Приложение A1.2. Система координат для моделирования дифракции

<!-- 260526Cl: 図(Coordinates4-5)に合わせ座標記号を数式(MathJax)化・用語色を図に一致 (.rp-*: docs/src/assets/stylesheets/extra.css)。 -->

Функция **Симулятор дифракции** моделирует дифракционную картину, зарегистрированную на детекторе. Детектор представляет собой конечную плоскость из пикселей, расположенную на фиксированном расстоянии от образца, и может быть наклонён относительно падающего пучка. Чтобы точно это воспроизвести, необходимы геометрическое соотношение между детектором и образцом, а также размер пикселя и число пикселей детектора. Об основной (ориентационной) системе координат см. [A1.1. Основная система координат и ориентация кристалла](1-orientation.md).

!!! note "Z и Y отличаются от ориентационной системы"
    В системе координат детектора <span class="rp-steel">$Z$</span> параллельна пучку, а <span class="rp-steel">$Y$</span> направлена вниз. Это отличается от ориентационной системы координат, в которой пучок идёт вдоль <span class="rp-blue">$-Z$</span>, а <span class="rp-green">$Y$</span> направлена вверх. Система детектора следует обычному соглашению для изображения/детектора (начало координат в верхнем левом углу, <span class="rp-steel">$Y$</span> возрастает вниз).

## До поворота (детектор перпендикулярен пучку)

![Система координат детектора при детекторе, перпендикулярном пучку](../../../assets/references/Coordinates4.png){width=500px}

Определяются три системы координат:

- <span class="rp-steel">**Реальные координаты** ($X$, $Y$, $Z$)</span> : трёхмерные декартовы координаты в мм с <span class="rp-steel">**образцом**</span> в качестве начала координат. <span class="rp-steel">$Z$</span> параллельна пучку; при взгляде вдоль <span class="rp-steel">$Z$</span> ось <span class="rp-steel">$X$</span> направлена вправо, а <span class="rp-steel">$Y$</span> — вниз. Когда детектор перпендикулярен пучку, <span class="rp-steel">$X$ / $Y$</span> параллельны <span class="rp-brown">$X'$ / $Y'$</span>.
- <span class="rp-brown">**Координаты детектора** ($X'$, $Y'$)</span> : двумерные координаты в мм на плоскости детектора с <span class="rp-brown">**foot**</span> в качестве начала координат. <span class="rp-brown">$X'$ / $Y'$</span> направлены на детекторе вправо / вниз и параллельны <span class="rp-cyan">$X''$ / $Y''$</span>.
- <span class="rp-cyan">**Пиксельные координаты** ($X''$, $Y''$)</span> : двумерные координаты в пикселях с <span class="rp-cyan">**верхним левым углом**</span> детектора в качестве начала координат, вдоль строк и столбцов пикселей детектора.

Когда детектор перпендикулярен пучку, <span class="rp-brown">**foot**</span> и <span class="rp-red">**direct spot**</span> совпадают, и <span class="rp-red">**Camera length 1**</span> равна <span class="rp-brown">**Camera length 2**</span>.

## После поворота (наклонённый детектор)

![Система координат детектора при наклонённом детекторе](../../../assets/references/Coordinates5.png){width=500px}

Наклон детектора описывается двумя параметрами:

| Параметр | Описание |
|-----------|-------------|
| <span class="rp-grass">$\varphi$</span> | Направление <span class="rp-grass">оси вращения</span> — её угол относительно оси <span class="rp-steel">$X$</span>, измеренный в плоскости <span class="rp-steel">$XY$</span> (<span class="rp-steel">$Z$</span> = 0) |
| <span class="rp-grass">$\tau$</span> | Угол поворота вокруг этой оси (правый винт) |

После наклона детектора:

- <span class="rp-red">**direct spot**</span> и <span class="rp-brown">**foot**</span> больше не совпадают.
- <span class="rp-red">**Camera length 1** ($C_1$)</span> = расстояние от <span class="rp-steel">образца</span> до <span class="rp-red">direct spot</span>.
- <span class="rp-brown">**Camera length 2** ($C_2$)</span> = расстояние от <span class="rp-steel">образца</span> до <span class="rp-brown">foot</span>.
- Начало <span class="rp-brown">**координат детектора**</span> остаётся в <span class="rp-brown">**foot**</span>; начало <span class="rp-cyan">**пиксельных координат**</span> остаётся в <span class="rp-cyan">**верхнем левом углу**</span>.
- Направления <span class="rp-steel">$X$ / $Y$</span> больше не совпадают с <span class="rp-brown">$X'$ / $Y'$</span>.

## Глоссарий параметров

| Термин | Определение |
|------|------------|
| <span class="rp-steel">**Образец (Sample)**</span> | Материал, рассеивающий падающий пучок; начало реальных координат |
| <span class="rp-steel">**Реальные координаты** ($X$, $Y$, $Z$)</span> | Трёхмерные координаты (мм) экспериментальной установки; начало в образце, <span class="rp-steel">$Z$</span> всегда параллельна пучку |
| <span class="rp-red">**Direct spot**</span> | Точка пересечения падающего пучка с детектором |
| <span class="rp-brown">**Foot**</span> | Основание перпендикуляра, опущенного из образца на плоскость детектора; начало координат детектора. Совпадает с direct spot только тогда, когда детектор перпендикулярен пучку. В режиме наложения изображения положение foot задаётся в пиксельных координатах |
| <span class="rp-brown">**Координаты детектора** ($X'$, $Y'$)</span> | Двумерные координаты (мм) на плоскости детектора; начало в foot |
| <span class="rp-cyan">**Пиксельные координаты** ($X''$, $Y''$)</span> | Двумерные координаты (пиксели) на плоскости детектора; начало в верхнем левом углу |
| <span class="rp-red">**Camera length 1** ($C_1$)</span> | Расстояние от образца до direct spot (мм) |
| <span class="rp-brown">**Camera length 2** ($C_2$)</span> | Расстояние от образца до foot (мм) |
| **Pixel size** | Длина стороны одного (квадратного) пикселя (мм); поддерживаются только квадратные пиксели |
| **Detector width / height** | Число пикселей по горизонтали / вертикали |
