# 键盘与鼠标快捷键

ReciPro 将许多功能绑定到 **组合键** 以及 **修饰键配合鼠标按键** 上 —— 这些操作在按钮或菜单上是看不到的。本页将它们全部汇总在一处。每个窗口自己的页面也会在开头附近重复列出其快捷键。

<kbd>F1</kbd> 在 **每个** 窗口都有效，会打开该窗口在本在线手册中的页面。

---

## 应用程序级快捷键

这些快捷键由[主窗口](0-main-window.md)注册，但在结构查看器、极射赤平投影、衍射模拟器、Spot ID 和计算器窗口获得焦点时仍然有效。

| 快捷键 | 操作 |
|----------|--------|
| <kbd>CTRL</kbd>+<kbd>SHIFT</kbd>+<kbd>D</kbd> | 开关衍射模拟器 |
| <kbd>CTRL</kbd>+<kbd>SHIFT</kbd>+<kbd>V</kbd> | 开关结构查看器 |
| <kbd>CTRL</kbd>+<kbd>SHIFT</kbd>+<kbd>S</kbd> | 开关极射赤平投影 |
| <kbd>CTRL</kbd>+<kbd>SHIFT</kbd>+<kbd>T</kbd> | 开关 Spot ID |
| <kbd>CTRL</kbd>+<kbd>SHIFT</kbd> + 方向键 | 将晶体沿该方向旋转一步（同时按住两个方向键可斜向旋转） |
| 快速双击 <kbd>CTRL</kbd> | 开关计算器 |
| <kbd>CTRL</kbd>+<kbd>SHIFT</kbd>+<kbd>R</kbd> | 切换所选晶体的 *Reserved* 标志 |
| <kbd>CTRL</kbd>+<kbd>ALT</kbd>+<kbd>SHIFT</kbd>+<kbd>C</kbd> | 截取 GUI 的屏幕截图（开发者工具；请先启用 **Capture GUI Components**） |
| ReciPro 启动时按住 <kbd>CTRL</kbd> | 以禁用 OpenGL 的方式启动（用于恢复图形问题） |

## 共用的交互模型

ReciPro 中几乎每个可交互视图都属于三大系列之一。只要知道它属于哪个系列，无需逐个窗口记忆，就能掌握其拖动/缩放行为。

### 3-D OpenGL 视图 { #3d }

由[结构查看器](5-structure-viewer.md)、[旋转几何](4-rotation-geometry.md)、[极射赤平投影](6-stereonet.md)的 3-D 球、[电子轨迹](8-electron-trajectory.md)以及 [EBSD](12-ebsd-simulation.md) 的几何 / master pattern 视图使用。

| 操作 | 结果 |
|--------|--------|
| 左键拖动 | 旋转 —— 靠近中心为轨迹球旋转，靠近边缘为面内滚转 |
| 右键上下拖动，或鼠标滚轮 | 缩放 |
| 中键拖动 | 平移（仅在启用处） |
| <kbd>CTRL</kbd> + 右键上下拖动 | 改变相机距离（仅透视模式） |
| <kbd>CTRL</kbd> + 右键双击 | 在正交 / 透视投影之间切换 |

个别窗口可能关闭平移或缩放（例如电子轨迹和 EBSD 的 3-D 视图禁用了平移）。

### 衍射图样视图 { #pattern }

由[衍射模拟器](7-diffraction-simulator/index.md)的图样、[EBSD](12-ebsd-simulation.md) 的菊池图样以及 2-D [极射赤平投影](6-stereonet.md)使用。与 3-D 视图的关键区别在于：**拖动旋转的是晶体本身**，而不仅仅是相机，因此每个联动窗口都会一起更新。

| 操作 | 结果 |
|--------|--------|
| 在中心附近左键拖动 | 倾斜晶体 |
| 在外围区域左键拖动 | 绕视线/束轴旋转晶体 |
| 右键单击 | 缩小 |
| 右键拖出一个框 | 放大到所选区域 |
| 中键拖动 | 平移 |

这些视图上**没有**鼠标滚轮缩放。

### 图像视图 { #image }

由 [HRTEM/STEM](9-hrtem-stem-simulator/index.md) 的结果窗格、[Spot ID v2](11-spot-id-v2.md) 的图像以及 [EBSD](12-ebsd-simulation.md) 的 2-D master pattern 使用。

| 操作 | 结果 |
|--------|--------|
| 左键拖动 / 中键拖动 | 平移 |
| 鼠标滚轮上 / 下 | 在光标处放大（×2）/ 缩小（×0.5） |
| 右键拖出一个框 | 放大到所选区域 |
| 右键单击 / 右键双击 | 缩小（×0.5） |

## 各窗口参考

### 0. 主窗口
[打开页面 →](0-main-window.md) · 另请参阅上面的应用程序级快捷键。

| 快捷键 | 操作 |
|----------|--------|
| 左键拖动方位小部件（左下角） | 旋转晶体 |
| 右键双击方位小部件 | 将小部件图像复制到剪贴板 |
| 单击 / 双击功能按钮 | 开关该窗口 / 将其强制置于最前 |
| 右键单击列表中的某个晶体 | 上下文菜单（重命名 / 生成副本 / 删除 / 导出 CIF…） |
| 双击 **Current Index** 标签 | 显示 / 隐藏最大 UVW 框 |
| 拖放一个文件 | 加载晶体列表（`.xml`、`.cdb2`）或晶体（`.cif`、`.amc`） |

### 1. 晶体数据库
[打开页面 →](1-crystal-database.md)

| 快捷键 | 操作 |
|----------|--------|
| 在搜索框中按 <kbd>ENTER</kbd> | 执行搜索 |
| 单击结果行 | 加载该晶体 |
| 在周期表弹出窗口中单击某个元素 | 循环切换其筛选条件：忽略 → 必须包含 → 必须排除 |

### 2. 对称性信息 · 3. 射束相互作用
对称性信息没有特殊的键盘/鼠标组合。在射束相互作用中，除了 <kbd>F1</kbd> 和 **Copy** 按钮外，还可拖动 **Scattering factors** 图上的竖直光标，以读取各元素的数值。
[对称性信息 →](2-symmetry-information.md) · [射束相互作用 →](3-beam-interaction.md)

### 4. 旋转几何
[打开页面 →](4-rotation-geometry.md) —— 六个**联动**的 [3-D 视图](#3d)；旋转其中任意一个都会让六个一起旋转。小的 *Axes* / *Objects* 视图禁用了缩放和平移。

### 5. 结构查看器
[打开页面 →](5-structure-viewer.md) —— 主视图是 [3-D 视图](#3d)。

| 快捷键 | 操作 |
|----------|--------|
| <kbd>CTRL</kbd>+<kbd>SHIFT</kbd>+<kbd>C</kbd> | 将渲染的图像复制到剪贴板 |
| 左键双击某个原子 | 显示坐标、最近邻距离和键角 |
| 左键拖动晶轴 gizmo | 旋转模型（无面内滚转） |
| 左键拖动光源 gizmo | 改变照明方向 |

### 6. 极射赤平投影
[打开页面 →](6-stereonet.md) —— 2-D 网是[衍射图样视图](#pattern)；可选的 3-D 球是 [3-D 视图](#3d)。

| 快捷键 | 操作 |
|----------|--------|
| 左键双击网 | 在 **Plane** 与 **Axis** 投影之间切换 |
| 在网上移动鼠标 | 读取光标下的 (hkl)/[uvw] |

### 7. 衍射模拟器
[打开页面 →](7-diffraction-simulator/index.md) —— 图样是[衍射图样视图](#pattern)（无滚轮缩放）。

| 快捷键 | 操作 |
|----------|--------|
| 左键双击某个衍射点 | 显示反射详情（指数、*d*、结构因子、偏离矢量） |
| <kbd>CTRL</kbd> + 中键拖动 | 移动探测器中心（显示探测器区域时） |
| 右键双击状态栏 | 复制当前设置的文本摘要 |
| 右键双击点亮的图层按钮（Spots / Kikuchi / Debye / Scale） | 使该图层闪烁开关 |
| 左键双击极射赤平投影 —— **TEM 样品台** 窗口 | 将样品台倾斜设到该点 |
| 方向键 —— **TEM 样品台** 窗口 | 逐步调整样品台倾斜（请先勾选 **Arrow keys**） |
| 拖放 `.prm` / 图像 —— **Detector geometry**，或 `.txt` —— **Dynamic compression** | 加载该数据 |

### 8. 电子轨迹
[打开页面 →](8-electron-trajectory.md) —— 一个禁用平移的 [3-D 视图](#3d)。

### 9. HRTEM / STEM 模拟器
[打开页面 →](9-hrtem-stem-simulator/index.md) —— 结果窗格是[图像视图](#image)，并一起平移/缩放。

| 快捷键 | 操作 |
|----------|--------|
| <kbd>CTRL</kbd>+<kbd>C</kbd>（图像网格获得焦点时） | 将图像作为元文件复制到剪贴板 |
| <kbd>CTRL</kbd> + 右键拖出一个框 | 选择一个矩形区域 |
| 左键双击某个窗格 | 最大化该窗格 / 还原网格（多窗格布局） |

### 10. Spot ID v1
[打开页面 →](10-spot-id.md) —— 该图像仅供参考（不可交互）。

| 快捷键 | 操作 |
|----------|--------|
| 双击结果列表中的某行 | 选择该晶体并将其旋转到匹配的晶带轴 |

### 11. Spot ID v2
[打开页面 →](11-spot-id-v2.md) —— 该图像是[图像视图](#image)，并在其上叠加衍射点编辑。

| 快捷键 | 操作 |
|----------|--------|
| 左键双击图像 | 添加一个衍射点（经峰拟合） |
| <kbd>CTRL</kbd> + 左键双击 | 添加一个衍射点并将其标记为直射 (000) 束 |
| 左键单击某个衍射点 | 选择最近的衍射点 |
| <kbd>CTRL</kbd> + 右键单击某个衍射点 | 删除最近的衍射点 |
| <kbd>CTRL</kbd> + 方向键 | 将所选衍射点移动一个像素 |
| 双击某个衍射点的行标题 | 缩放到该衍射点（×2） |

### 12. EBSD 模拟
[打开页面 →](12-ebsd-simulation.md) —— 菊池图样是[衍射图样视图](#pattern)；3-D 视图是 [3-D 视图](#3d)（平移关闭）；2-D master pattern 是[图像视图](#image)。

| 快捷键 | 操作 |
|----------|--------|
| 双击菊池图样 | 选取光标下的探测器子单元并显示其统计信息 |

### 20. 宏
[打开页面 →](20-macro/index.md)

| 快捷键 | 操作 |
|----------|--------|
| <kbd>CTRL</kbd>+<kbd>S</kbd> | 将编辑器文本保存回所选的宏列表条目 |
| <kbd>F10</kbd> | 前进一步（在逐步执行期间） |
| 双击函数帮助列表中的某行 | 在插入点处插入该函数的签名 |
| 拖放一个 `.mcr` 文件 | 将其加载到编辑器中 |
