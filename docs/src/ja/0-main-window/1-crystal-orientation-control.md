# 結晶方位の設定

ReciPro では、[メインウィンドウ](index.md) で選択中の結晶を対話的に回転させることができます。ここで設定した結晶方位は、回折シミュレータ・Structure Viewer・ステレオネット・HRTEM/STEM シミュレータ・EBSD シミュレータなど他のウィンドウでも共通して使われます。

> 短い操作動画は [How to use（動画）](../appendix/a0-how-to-use.md) のページにあります。

---

## 結晶方位を設定する基本的な方法

| 方法 | こんなとき | 操作場所 |
|------|-----------|---------|
| マウスドラッグ | 結晶軸を見ながら自由に回転させたいとき | 「現在の結晶方位」パネル |
| 矢印ボタン | 小さな回転を繰り返し行いたいとき | 「回転」パネル |
| 晶帯軸 | \([001]\) や \([110]\) など、見たい方向が分かっているとき | 「軸/面投影」/ 晶帯軸の入力 |
| 面法線 | 結晶面 \((hkl)\) をスクリーンに垂直にしたいとき | 「軸/面投影」/ 面の入力 |
| オイラー角 | 再現可能な数値で方位を指定したいとき | 「オイラー角」 |

---

## 推奨ワークフロー

1. **結晶リスト**で結晶を選択します。
2. **軸/面投影**で、目的の晶帯軸または面法線をスクリーンに垂直に設定します。
3. 面内方向が重要な場合は、晶帯軸方向と面方向の両方を設定し、画面上向きの方向を固定します。
4. [回折シミュレータ](../7-diffraction-simulator/index.md)、[Structure Viewer](../5-structure-viewer.md)、[HRTEM/STEM シミュレータ](../9-hrtem-stem-simulator/index.md) など、目的のツールを開きます。

ここに表示される結晶方位は、単なる表示上の設定ではありません。シミュレーションで用いる入射ビーム方向と結晶座標系の関係を定義します。
