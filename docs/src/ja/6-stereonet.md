# ステレオネット (Stereonet)

**ステレオネット** は、結晶面や晶帯軸の方向をステレオネット投影で表示します。

![ステレオネット](../assets/cap-ja-auto/FormStereonet.png)

---

## メインエリア

選択した結晶の結晶面・晶帯軸のステレオネット投影が表示されます。

### マウス操作

| 操作 | 動作 |
|------|------|
| 左ドラッグ | 結晶を回転 |
| 右クリック | ズームアウト |
| 右ドラッグ / ホイール | ズームイン |
| ダブルクリック | 面/軸モードの切替 |
| 中ドラッグ | 平行移動 |

マウスを投影上で動かすと、**カーソル位置**の面/軸指数が表示されます。

---

## ファイルメニュー

表示中のステレオネットを保存またはコピーします。**ラスタ形式**または**ベクタ形式**で出力可能です。ベクタ形式でPowerPointに貼り付ければ、テキストのフォントや線の太さを自由に変更できます。

---

## モード

![モード](../assets/cap-ja-auto/FormStereonet.panel3.groupBoxMode.png)

### 投影対象
**結晶軸**（晶帯軸 [*uvw*]）・**結晶面**（{*hkl*}）・**菊池線**（菊池線ペア）を選択。指数ラベルの表示、構造因子による重み付けも設定できます。

### 投影法

| 投影法 | 説明 |
|--------|------|
| **等角投影 (Wulff)** | 角度関係を保存するが、立体角（面積）は保存しない |
| **等積投影 (Schmidt)** | 立体角（面積）を保存するが、角度関係は保存しない |

**描画半球**で上半球／下半球を選択できます。

> 三方晶・六方晶系では、メインウィンドウの **オプション ▸ Use Miller-Bravais (hkil) index** でミラー・ブラベー（4指数）表記に切り替えられます。

---

## 指数 (Indices)

![指数](../assets/cap-ja-auto/FormStereonet.panel3.groupBoxIndices.png)

描画する結晶面/晶帯軸を設定します。

### 範囲モード
[*u v w*] または {*h k l*} 指数の範囲を指定。

### 指定モード
特定の軸や面を個別に指定。指数を入力後 **追加** で追加、**削除** で削除。**等価な指数を含める** をチェックすると、結晶学的に等価なすべての面/軸を描画します。

**色** を設定できます。**色を自動で変更する** をチェックすると、等価な軸・面の組ごとに自動で色分けされます。

## タブメニュー

### 表示

![表示タブ](../assets/cap-ja-auto/FormStereonet.tabControl.tabPage1.png)

| 設定 | 説明 |
|------|------|
| 文字（サイズ） | 指数ラベルのサイズ |
| 点（サイズ） | 投影点のサイズ |
| 色 | 点・ラベル・ステレオネット外枠の色 |
| 輪郭 | ステレオネット外枠の表示オプション |

### 大円

![大円タブ](../assets/cap-ja-auto/FormStereonet.tabControl.tabPage2.png)

大円や小円を描画します。**晶帯軸の指数** または **晶帯に属する2つの結晶面** で指定します。

### 線源

![線源タブ](../assets/cap-ja-auto/FormStereonet.tabControl.tabPage4.png)

投影に用いる線源 (X線 / 電子線 / 中性子線) と波長 (またはエネルギー) を設定します。
