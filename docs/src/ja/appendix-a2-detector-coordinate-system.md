# 付録 A2. 検出器座標系の定義

**Crystal Diffraction** は検出器上の回折パターンをシミュレーションします。検出器はピクセルの集合で構成された有限サイズの平面で、試料から一定距離に配置され、入射ビームに対して傾いている場合もあります。

---

## 回転前（検出器がビームに垂直）

### 実座標 (X, Y, Z)
mm単位の3次元直交座標。**試料**を原点とする。
- **Z軸**: 常にビーム方向に平行（付録A1の定義とは異なることに注意）
- **X軸**: Z軸に向かって右方向
- **Y軸**: 下方向

### 検出器座標 (X', Y')
検出器平面上のmm単位の2次元直交座標。**foot**を原点とする。X'とY'はX''とY''に平行。

### ピクセル座標 (X'', Y'')
ピクセル単位の2次元座標。検出器の**左上隅**を原点とする。

### 主要な定義

| 用語 | 定義 |
|------|------|
| **Foot** | 試料から検出器平面への垂線の足。検出器がビームに垂直の場合、footと透過スポットは一致 |
| **透過スポット (Direct spot)** | 入射ビームと検出器の交点 |
| **Camera length 2 (C2)** | 試料からfootまでの距離 (mm) |
| **Camera length 1 (C1)** | 試料から透過スポットまでの距離 (mm) |
| **Pixel size** | 1ピクセルの一辺の長さ (mm)。正方ピクセルのみ |
| **Detector width/height** | 水平/垂直方向のピクセル数 |

---

## 回転後（検出器が傾いた場合）

検出器の傾きを2つのパラメータで表現します：

| パラメータ | 説明 |
|-----------|------|
| **φ** | 回転軸の方向（XY平面上でX軸からの角度） |
| **τ** | φで定義された軸周りの回転角（右ネジの方向） |

回転後：
- **透過スポット**と**foot**は一致しなくなる
- **C1** = 試料から透過スポットまでの距離
- **C2** = 試料からfootまでの距離
- 検出器座標の原点は常に**foot**
- ピクセル座標の原点は常に**左上隅**
- 傾いた場合、X/Y方向はX'/Y'方向と一致しない
