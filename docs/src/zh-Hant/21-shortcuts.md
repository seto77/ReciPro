# 鍵盤與滑鼠快速鍵

ReciPro 將許多功能繫結到 **組合鍵** 以及 **修飾鍵搭配滑鼠按鍵** 上 —— 這些操作在按鈕或選單上是看不到的。本頁將它們全部彙整在一處。每個視窗自己的頁面也會在開頭附近重複列出其快速鍵。

<kbd>F1</kbd> 在 **每個** 視窗都有效，會開啟該視窗在本線上手冊中的頁面。

---

## 應用程式層級快速鍵

這些快速鍵由[主視窗](0-main-window.md)註冊，但在結構檢視器、極網、繞射模擬器、Spot ID 與計算機視窗取得焦點時仍然有效。

| 快速鍵 | 操作 |
|----------|--------|
| <kbd>CTRL</kbd>+<kbd>SHIFT</kbd>+<kbd>D</kbd> | 開關繞射模擬器 |
| <kbd>CTRL</kbd>+<kbd>SHIFT</kbd>+<kbd>V</kbd> | 開關結構檢視器 |
| <kbd>CTRL</kbd>+<kbd>SHIFT</kbd>+<kbd>S</kbd> | 開關極網 |
| <kbd>CTRL</kbd>+<kbd>SHIFT</kbd>+<kbd>T</kbd> | 開關 Spot ID |
| <kbd>CTRL</kbd>+<kbd>SHIFT</kbd> + 方向鍵 | 將晶體沿該方向旋轉一步（同時按住兩個方向鍵可斜向旋轉） |
| 快速雙擊 <kbd>CTRL</kbd> | 開關計算機 |
| <kbd>CTRL</kbd>+<kbd>SHIFT</kbd>+<kbd>R</kbd> | 切換所選晶體的 *Reserved* 標記 |
| <kbd>CTRL</kbd>+<kbd>ALT</kbd>+<kbd>SHIFT</kbd>+<kbd>C</kbd> | 擷取 GUI 的螢幕擷圖（開發者工具；請先啟用 **Capture GUI Components**） |
| ReciPro 啟動時按住 <kbd>CTRL</kbd> | 以停用 OpenGL 的方式啟動（用於修復圖形問題） |

## 共用的互動模型

ReciPro 中幾乎每個可互動檢視都屬於三大系列之一。只要知道它屬於哪個系列，無需逐個視窗記憶，就能掌握其拖曳/縮放行為。

### 3-D OpenGL 檢視 { #3d }

由[結構檢視器](5-structure-viewer.md)、[旋轉幾何](4-rotation-geometry.md)、[極網](6-stereonet.md)的 3-D 球、[電子軌跡](8-electron-trajectory.md)以及 [EBSD](12-ebsd-simulation.md) 的幾何 / master pattern 檢視使用。

| 操作 | 結果 |
|--------|--------|
| 左鍵拖曳 | 旋轉 —— 靠近中心為軌跡球旋轉，靠近邊緣為面內滾轉 |
| 右鍵上下拖曳，或滑鼠滾輪 | 縮放 |
| 中鍵拖曳 | 平移（僅在啟用處） |
| <kbd>CTRL</kbd> + 右鍵上下拖曳 | 改變相機距離（僅透視模式） |
| <kbd>CTRL</kbd> + 右鍵雙擊 | 在正交 / 透視投影之間切換 |

個別視窗可能關閉平移或縮放（例如電子軌跡和 EBSD 的 3-D 檢視停用了平移）。

### 繞射圖樣檢視 { #pattern }

由[繞射模擬器](7-diffraction-simulator/index.md)的圖樣、[EBSD](12-ebsd-simulation.md) 的菊池圖樣以及 2-D [極網](6-stereonet.md)使用。與 3-D 檢視的關鍵區別在於：**拖曳旋轉的是晶體本身**，而不僅是相機，因此每個連動視窗都會一起更新。

| 操作 | 結果 |
|--------|--------|
| 在中心附近左鍵拖曳 | 傾斜晶體 |
| 在外圍區域左鍵拖曳 | 繞視線/束軸旋轉晶體 |
| 右鍵點按 | 縮小 |
| 右鍵拖出一個框 | 放大到所選區域 |
| 中鍵拖曳 | 平移 |

這些檢視上**沒有**滑鼠滾輪縮放。

### 影像檢視 { #image }

由 [HRTEM/STEM](9-hrtem-stem-simulator/index.md) 的結果窗格、[Spot ID v2](11-spot-id-v2.md) 的影像以及 [EBSD](12-ebsd-simulation.md) 的 2-D master pattern 使用。

| 操作 | 結果 |
|--------|--------|
| 左鍵拖曳 / 中鍵拖曳 | 平移 |
| 滑鼠滾輪上 / 下 | 在游標處放大（×2）/ 縮小（×0.5） |
| 右鍵拖出一個框 | 放大到所選區域 |
| 右鍵點按 / 右鍵雙擊 | 縮小（×0.5） |

## 各視窗參考

### 0. 主視窗
[開啟頁面 →](0-main-window.md) · 另請參閱上面的應用程式層級快速鍵。

| 快速鍵 | 操作 |
|----------|--------|
| 左鍵拖曳方位小工具（左下角） | 旋轉晶體 |
| 右鍵雙擊方位小工具 | 將小工具影像複製到剪貼簿 |
| 單擊 / 雙擊功能按鈕 | 開關該視窗 / 將其強制置於最前 |
| 右鍵點按清單中的某個晶體 | 內容功能表（重新命名 / 複製 / 刪除 / 匯出 CIF…） |
| 雙擊 **Current Index** 標籤 | 顯示 / 隱藏最大 UVW 框 |
| 拖放一個檔案 | 載入晶體清單（`.xml`、`.cdb2`）或晶體（`.cif`、`.amc`） |

### 1. 晶體資料庫
[開啟頁面 →](1-crystal-database.md)

| 快速鍵 | 操作 |
|----------|--------|
| 在搜尋欄位中按 <kbd>ENTER</kbd> | 執行搜尋 |
| 點按結果列 | 載入該晶體 |
| 在週期表彈出視窗中點按某個元素 | 循環切換其篩選條件：忽略 → 必須包含 → 必須排除 |

### 2. 對稱性資訊 · 3. 電子束交互作用
對稱性資訊沒有特殊的鍵盤/滑鼠組合。在電子束交互作用中，除了 <kbd>F1</kbd> 和 **複製** 按鈕外，還可拖曳 **散射因子** 圖上的垂直游標，以讀取各元素的數值。
[對稱性資訊 →](2-symmetry-information.md) · [電子束交互作用 →](3-beam-interaction.md)

### 4. 旋轉幾何
[開啟頁面 →](4-rotation-geometry.md) —— 六個**連動**的 [3-D 檢視](#3d)；旋轉其中任一個都會讓六個一起旋轉。小的 *Axes* / *Objects* 檢視停用了縮放和平移。

### 5. 結構檢視器
[開啟頁面 →](5-structure-viewer.md) —— 主檢視是 [3-D 檢視](#3d)。

| 快速鍵 | 操作 |
|----------|--------|
| <kbd>CTRL</kbd>+<kbd>SHIFT</kbd>+<kbd>C</kbd> | 將算繪的影像複製到剪貼簿 |
| 左鍵雙擊某個原子 | 顯示座標、最近鄰距離和鍵角 |
| 左鍵拖曳晶軸操作器 | 旋轉模型（無平面內自旋） |
| 左鍵拖曳光源操作器 | 改變照明方向 |

### 6. 極網
[開啟頁面 →](6-stereonet.md) —— 2-D 網是[繞射圖樣檢視](#pattern)；可選的 3-D 球是 [3-D 檢視](#3d)。

| 快速鍵 | 操作 |
|----------|--------|
| 左鍵雙擊網 | 在 **Plane** 與 **Axis** 投影之間切換 |
| 在網上移動滑鼠 | 讀取游標下的 (hkl)/[uvw] |

### 7. 繞射模擬器
[開啟頁面 →](7-diffraction-simulator/index.md) —— 圖樣是[繞射圖樣檢視](#pattern)（無滾輪縮放）。

| 快速鍵 | 操作 |
|----------|--------|
| 左鍵雙擊某個繞射點 | 顯示反射詳情（指數、*d*、結構因子、偏離向量） |
| <kbd>CTRL</kbd> + 中鍵拖曳 | 移動偵測器中心（顯示偵測器區域時） |
| 右鍵雙擊狀態列 | 複製目前設定的文字摘要 |
| 右鍵雙擊點亮的圖層按鈕（Spots / Kikuchi / Debye / Scale） | 使該圖層閃爍開關 |
| 左鍵雙擊極網 —— **TEM holder** 視窗 | 將試樣台傾斜設到該點 |
| 方向鍵 —— **TEM holder** 視窗 | 逐步調整試樣台傾斜（請先勾選 **Arrow keys**） |
| 拖放 `.prm` / 影像 —— **Detector geometry**，或 `.txt` —— **Dynamic compression** | 載入該資料 |

### 8. 電子軌跡
[開啟頁面 →](8-electron-trajectory.md) —— 一個停用平移的 [3-D 檢視](#3d)。

### 9. HRTEM / STEM 模擬器
[開啟頁面 →](9-hrtem-stem-simulator/index.md) —— 結果窗格是[影像檢視](#image)，並一起平移/縮放。

| 快速鍵 | 操作 |
|----------|--------|
| <kbd>CTRL</kbd>+<kbd>C</kbd>（影像格線取得焦點時） | 將影像作為中繼檔複製到剪貼簿 |
| <kbd>CTRL</kbd> + 右鍵拖出一個框 | 選取一個矩形區域 |
| 左鍵雙擊某個窗格 | 最大化該窗格 / 還原格線（多窗格版面） |

### 10. Spot ID v1
[開啟頁面 →](10-spot-id.md) —— 該影像僅供參考（不可互動）。

| 快速鍵 | 操作 |
|----------|--------|
| 雙擊結果清單中的某列 | 選取該晶體並將其旋轉到相符的晶帶軸 |

### 11. Spot ID v2
[開啟頁面 →](11-spot-id-v2.md) —— 該影像是[影像檢視](#image)，並在其上疊加繞射點編輯。

| 快速鍵 | 操作 |
|----------|--------|
| 左鍵雙擊影像 | 新增一個繞射點（經峰擬合） |
| <kbd>CTRL</kbd> + 左鍵雙擊 | 新增一個繞射點並將其標記為直射 (000) 束 |
| 左鍵點按某個繞射點 | 選取最近的繞射點 |
| <kbd>CTRL</kbd> + 右鍵點按某個繞射點 | 刪除最近的繞射點 |
| <kbd>CTRL</kbd> + 方向鍵 | 將所選繞射點移動一個像素 |
| 雙擊某個繞射點的列首 | 縮放到該繞射點（×2） |

### 12. EBSD 模擬
[開啟頁面 →](12-ebsd-simulation.md) —— 菊池圖樣是[繞射圖樣檢視](#pattern)；3-D 檢視是 [3-D 檢視](#3d)（平移關閉）；2-D master pattern 是[影像檢視](#image)。

| 快速鍵 | 操作 |
|----------|--------|
| 雙擊菊池圖樣 | 選取游標下的偵測器子格並顯示其統計資料 |
